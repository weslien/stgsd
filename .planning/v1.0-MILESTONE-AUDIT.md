### CLI-12 Detail

**Requirement:** "Installable to `~/.claude/bin/stclaude`"
**Actual:** Binary installed at `~/.claude/bin/stclaude.mjs` (with `.mjs` extension)
**Root cause:** Node.js ESM detection requires `.mjs` extension when file is outside a `package.json` scope with `"type": "module"`
**Impact:** If agents invoke `stclaude` without extension, binary won't be found. If agents use full path with `.mjs`, it works.
**Fix:** Add symlink `~/.claude/bin/stclaude -> stclaude.mjs` or wrapper script
